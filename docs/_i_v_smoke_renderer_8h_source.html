218 /**
219 * Multi-Volume Ray Marching with Occupancy Optimization (Three-Pass Pipeline).
220 * Uses tile-based occupancy grid for efficient empty space skipping.
221 * Processes all volumes with correct Beer-Lambert integration.
222 * Outputs to Dual Render Targets (Albedo + Mask) at reduced resolution.
223 *
224 * Pipeline:
225 * Pass 0: Tile Setup - Compute per-tile depth range and quick volume mask
226 * Pass 1: Occupancy Build - Build View and Light occupancy 3D textures
227 * Pass 2: Ray March - Use occupancy for efficient skipping
228 *
229 * @param GraphBuilder RDG builder
230 * @param View Current scene view
231 * @param RenderData Pre-packed render data (created on Game Thread)
232 * @param SmokeAlbedoTex UAV texture for smoke color output
233 * @param SmokeMaskTex UAV texture for smoke opacity mask
234 * @param TexSize Size of output textures (may be reduced resolution)
235 * @param ViewportSize Size of the full viewport for depth sampling
236 * @param ViewRectMin Offset into full scene texture for depth sampling
237 */

239 FRDGBuilder& GraphBuilder,
240 const FSceneView& View,

242 FRDGTextureRef SmokeAlbedoTex,

244 const FIntPoint& TexSize,
245 const FIntPoint& ViewportSize,
246 const FIntPoint& ViewRectMin
247 );
