10/**
11 * A primitive component that dynamically generates collision geometry based on the voxel grid data.
12 *
13 * ## Overview
14 * Unlike standard static meshes, this component constructs a set of box colliders (AggGeom)
15 * representing the active voxels. It uses a binary greedy meshing algorithm to merge adjacent voxels into
16 * larger boxes to minimize the physics cost.
17 *
18 * ## Usage
19 * This is primarily designed for query-only interactions, such as:
20 * - Blocking AI Line of Sight (Visibility channel).
21 * - Preventing camera clipping.
22 * - Simple projectile blocking.
23 *
24 * @note Frequent updates to collision geometry are expensive. Use `MinCollisionUpdateInterval`
25 * and `MinCollisionUpdateVoxelNum` to throttle updates.
26 */

54 * @param VoxelBitArray A bitmask buffer where each `uint64` element represents a row of voxels along the X-axis.

56 * @param GridResolution The resolution of the voxel grid (Width, Depth, Height).
