Detailed Description

Manages registered smoke volumes and handles rendering. Owns shared rendering resources (noise volume) and reads settings from UIVSmokeSettings.

Prepare render data from all registered volumes. Must be called on Game Thread. Copies and packs all volume data for safe Render Thread access. If volume count exceeds MaxSupportedVolumes (128), filters by distance from camera.
