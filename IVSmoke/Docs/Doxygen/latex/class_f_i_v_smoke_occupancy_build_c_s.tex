\doxysubsection{Detailed Description}

Dispatch:\+ (ceil(Tile\+Count\+X/\+8), ceil(Tile\+Count\+Y/\+8), ceil(Step\+Slice\+Count/\+4)) Each texel stores a uint4 bitmask (128 bits for 128 volumes).
