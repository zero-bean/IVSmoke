\doxysubsection{Detailed Description}
GPU-\/side volume metadata for single-\/pass multi-\/volume ray marching. Each volume has its own transform, bounds, and rendering parameters. This struct is uploaded to a Structured\+Buffer for GPU access.

Memory layout:\+ 256 bytes (aligned to 16-\/byte boundary)
