\doxysubsection{Detailed Description}
Global settings for \doxylink{class_i_v_smoke}{IVSmoke} plugin. Accessible via Project Settings \texorpdfstring{$>$}{>} Plugins \texorpdfstring{$>$}{>} \doxylink{class_i_v_smoke}{IVSmoke}.

These settings affect ALL smoke volumes globally. For per-\/volume appearance (color, density), use \doxylink{class_u_i_v_smoke_smoke_preset}{UIVSmoke\+Smoke\+Preset}.
