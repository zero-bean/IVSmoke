\doxysubsection{Detailed Description}
Data for a single shadow cascade. Contains render targets, matrices, and update state.\hypertarget{struct_f_i_v_smoke_cascade_data_autotoc_md2}{}\doxysubsection{\texorpdfstring{TIMING MODEL (Single-\/\+Buffer):\+}{TIMING MODEL (Single-\/\+Buffer):\+}}\label{struct_f_i_v_smoke_cascade_data_autotoc_md2}
This uses a SINGLE-\/\+BUFFER approach because Scene\+Capture\+Component2D with b\+Capture\+Every\+Frame=true executes capture in the SAME frame\textquotesingle{}s render pass.

Frame N timeline:\+
\begin{DoxyEnumerate}
\item Game Thread:\+ Update() calculates VP matrix and sets Capture\+Component transform
\item Render Thread:\+ Scene\+Capture\+Component2D captures depth using current transform
\item Render Thread:\+ Ray march shader samples the captured texture with VP matrix
\end{DoxyEnumerate}

Since capture and sampling happen in the SAME frame, no double-\/buffering is needed. The VP matrix calculated in Update() matches the texture captured in the same frame.

NOTE:\+ Double-\/buffering would be needed if capture happened at END of frame and \hypertarget{struct_f_i_v_smoke_cascade_data_autotoc_md3}{}\doxysubsection{\texorpdfstring{sampling happened at START of next frame, but that\textquotesingle{}s not the case here.}{sampling happened at START of next frame, but that\textquotesingle{}s not the case here.}}\label{struct_f_i_v_smoke_cascade_data_autotoc_md3}
